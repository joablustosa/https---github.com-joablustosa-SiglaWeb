Módulos do Sistema

Cada módulo tem uma funcionalidade onde o usuário realiza operações determinadas. Por exemplo, quando é necessário criar uma viagem, deve-se usar o módulo de Coordenação de Viagens.

Quando o usuário deseja definir a programação de um motorista, deve-se usar o módulo de Escala de Motoristas, e assim por diante. Essa interdependência é importante para que não tenha diversos cadastros de uma mesma operação em departamentos distintos da empresa.
